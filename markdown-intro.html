markdown编辑器测评

发表于 2017-10-09 | 阅读次数:
字数统计: 671 字 | 阅读时长 ≈ 2 min

markdown可以看作是一个快速排版的工具，可以让你在码字的同时无需额外点击其他按钮而实现的自动排版，你无需考虑字号字体颜色加粗等各种琐事，只需要敲入特定符号即可，而且没有那些乱七八糟的windows拼写检查的波浪线！！

下面是一些适用场合|局限性|各平台markdown编辑器测评

markdown上手之后几乎是可以替代任何非严肃环境下的使用word的场合：

  • 课堂作业、实验报告
  • 上传至自己的博客
  • 课堂笔记、读书笔记
  • 任何对字号字体等无严格要求的文档
  • whatever u want

局限性：

  • 居中
  • 多空行、多空格
  • 严格字号字体要求

Markdown编辑器推荐

太长不看版

平台 Editor
Windows Typora
Mac OS MWeb / Ulysses
Web 不推荐

基本需求

  • 基本markdown语法满足
  • 可导出html、pdf、png等格式
  • ⚠️支持图片拖拽上传
  • 可预览（所见即所得/左编辑右预览）
  • 代码高亮

Windows

Typora

Typora官网:www.typora.io

  • 约42MB
  • 所见即所得
  • 支持图片拖拽
  • 代码高亮
  • 支持latex公式

Atom

Atom官网:atom.io

  • 约132MB
  • 一个强大的编辑器
  • 适合比起所见即所得更偏好编辑+预览的旁友们

由于虚拟机配置问题本人并没有实际操作评测，目测Typora已经满足日常所需且atom配置markdown较为繁琐

仍然对此编辑器好奇的小猴子可前往👇🏽一探究竟

Atom配置markdown参考博客：https://sspai.com/post/40460

sublime text

  • 支持markdown的强大的文本编辑器
  • 需要添加扩展来实现代码高亮能功能
  • 不支持图片拖拽
  • 不支持预览，不过有在浏览器里预览的扩展

MarkdownPad

别想了

插入图片是pro高级功能

要付费的🙄

OS X

MWeb

  • 基本markdown语法满足
  • 可导出html、pdf、png等格式
  • 支持图片拖拽上传
  • 可实时预览
  • 文档库
  • 统计字数等杂七杂八的功能……
  • latex公式 $\sqrt{x}$

Ulysess

同MWeb

几乎都是完美der~

放心用就好了~

我的破解版会闪退，但不会丢失进度，不知道是不是个别原因

mac系统下以上两款均需付费，破解版请前往www.waitsun.com或其他途径自寻
请支持正版！
